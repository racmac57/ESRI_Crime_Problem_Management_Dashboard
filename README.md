# ESRI Crime Problem Management Dashboard

This repository supports the Hackensack Police Department Safe Streets Operations Control Center (HPD SSOCC) deployment of the Esri Crime Problem Management (CPM) solution version 2.0 on ArcGIS Online (AGOL). It collects training materials, dashboard configuration files, automation scripts, and distribution documentation that support analysts and command staff in creating, monitoring, and evaluating crime problems.

## Repo Structure

| Folder | Description |
|--------|-------------|
| `training_materials/` | Session notes, recordings, and step-by-step guides for analysts. |
| `dashboard_files/` | AGOL exports, configuration files, and screenshots (placeholder until operational). |
| `scripts/` | Python and ArcPy automation scripts for AGOL content analysis. |
| `docs/` | Generated training documentation prepared for distribution. |

## Current Phase

Training and Knowledge Transfer. The dashboard is not yet fully operational. Training materials, session notes, and documentation are being collected while AGOL content cleanup, version verification, and data source configuration are completed.
