# Project Summary

## What the Solution Is

The Crime Problem Management (CPM) solution version 2.0 is an Esri product deployed on ArcGIS Online (AGOL). It gives a police agency a structured way to define crime problems, associate incident and call data with those problems, document responses, and evaluate outcomes. HPD SSOCC is deploying CPM v2.0 to support data-driven problem-oriented policing.

## Key Components

### Crime Problem Manager (Experience Builder web app)

The primary analyst tool. Analysts use it to create and manage crime problems, document responses, and record evaluations. It is the system of record for the narrative and lifecycle of each problem.

### Crime Problem Dashboard

The command staff monitoring tool. It displays active problems, priorities, districts, and response status so command staff can track the operational picture at a glance.

### Crime Problem Analyst Desktop App

An ArcGIS Pro project with task-based workflows. Analysts use it to draw problem boundaries, associate incidents (crimes and calls for service) that fall within a boundary and time range, and conduct spatial analysis.
